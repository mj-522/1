워드프로세서의 정의


워드프로세서란? 문서작성 H/W와 S/W 워드프로세서를 위한 하드웨어 구성 압력 장치-ex)키보드, 마우스, 스캐너 표시 장치-ex)CRT, PDP 저장 장치-ex)하드디스크, CD-ROM, DVD 출력장치-ex)프린터, 플로터, COM 기억장치-주기억장치-ROM(비휘발성),RAM(휘발성) -보조기억장치-하드디스크, USB, CD/DVD 워드프로세서의 기본용어 강제 개행 -문자가 다 채워지지 않은 상태에서 다음 줄로 이동 자동 개행 -다 채워지면 다음 줄로 이동하는 기능 다단 -한 페이지를 여러 단으로 나눔 레이아웃(미리보기, 소프트 카피→모니터 출력) -적당한 위치에 균형있게 배치하는 기능 *위지윅(하드카피)→종이로 출력 래그드 -한쪽 끝이 정렬되어 있지 않은 상태 마진 -상,하,좌,우에 두는 공백 머지 -두개의 파일을 하나로 만드는 기능 색인 -중요한 용어를 쪽 번호와 함께 수록한 목록 센터링 -가운데 정렬 소트 -문서를 오름차순이나 내림차순으로 재배열 워드 랩 -단어를 자르지 않고 다음 줄로 넘기는 기능 영문 균등 -단어와 단어사이를 균등하게 배분함 옵션 -선택항목 캡션 -표나 그림에 붙이는 제목 클립아트 -잘라낸 그림, 그래픽 데이터 모음 피치 -1인치에 표시되는 문자수로 숫자가↑, 글자간격↓ 보일러 플레이트 -주석(보충설명, 각주, 미주) 커닝 -글자와 글자 사이의 간격을 미세하게 조정하는 작업 글자간격조정→자간 조정 클립보드 -버퍼, 임시기억장소, 복사, 이동, 캡쳐, 붙이기(=잘라내기) 입력 및 저장기능 저장하기 -자동저장(ASV) -삭제되거나 파괴되었을 경우 백업파일(BAK)을 생성 저장기능관련용어 세이브 -보조기억장치 로드 -주기억장치 키보드 기능 토글키 -한/영, Insert, Caps lock, num lock, scroll lock 조합키 -shift, ctrl, alt 바로가기 키 -ctrl+c(복사하기) -ctrl+x(오려두기) -ctrl+v(붙이기)
